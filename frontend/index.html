    Workflow guide

    How to use AnomlyX

    1. upload_file
      Upload an inspection image

      Select a JPG, PNG, or WEBP image. The API sends it to the model and applies the predicted defect and severity.

    2. tune
      Adjust manual inputs

      Use the defect selector, severity controls, and reference thumbnails when you need to correct or test the diagnosis manually.

    3. edit_note
      Add inspection details

      Fill inspector, batch, material, location, and notes so the report has the right production context.

    4. check_circle
      Generate the diagnosis

      Click Generate Diagnosis to open the report with the selected image, root cause, remedy, and prevention checklist.

    5. print
      Print or review the report

      Use Print on the report page, or go back to Diagnosis to revise the inputs and regenerate the report.
